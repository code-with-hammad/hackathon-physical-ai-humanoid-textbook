# Data Model: Premium Landing Page

This feature is primarily presentational, and its data is derived from the existing file structure of the Docusaurus site at build time. There is no new persistent data model.

## Module/Chapter Structure

The data structure for the side navigation will be an array of `Module` objects, generated by traversing the `docs/` directory.

### Module

Represents a top-level module or category in the book.

- **`title`** (string): The display name of the module (e.g., "Digital Twin Gazebo Unity"). This will be derived from the directory name or a `_category_.json` file.
- **`icon`** (string, optional): A placeholder for the module's icon.
- **`chapters`** (array of `Chapter`): A list of chapters within the module.

### Chapter

Represents a single chapter or document within a module.

- **`title`** (string): The display name of the chapter. This will be derived from the markdown file's front matter (`title`) or the file name.
- **`path`** (string): The URL path to the chapter (e.g., `/docs/digital-twin-gazebo-unity/installation`).
