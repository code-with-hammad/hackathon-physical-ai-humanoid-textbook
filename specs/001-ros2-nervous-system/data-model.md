# Data Model: Module and Chapter Structure

## Module

- **`title`**: The title of the module (e.g., "Module 1: The Robotic Nervous System (ROS 2)")
- **`chapters`**: A list of chapters in the module.

## Chapter

- **`title`**: The title of the chapter (e.g., "Chapter 1: ROS 2 Nodes, Topics, and Services")
- **`content`**: The content of the chapter in Markdown format.
